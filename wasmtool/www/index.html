Dreamy's DMM Auto-Tools

This is a "static site", it works only within your browser, and you don't download any .exe files. This page cannot access anything that you don't give it access to. Browsers generally are safe, or at least a million times safer than downloading random .exe files from the internet. Full code is here.

Features

Replaces manifold4w pipes with appropriate w3/straight/angled pipes. Only works for supply, scrubber, aux, fuel pipes.

Usage

  1. make a commit so you don't lose any work if this fails
  2. select options, see instructions
  3. prepare the map like instructed in the selected options
  4. click the button below to select a dmm file to process
  5. wait a few seconds
  6. save the resulting dmm file
